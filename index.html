Stereoscopic Targetting Project

Robotic Football Club

In the sport of Robotic Football, completing a pass from the quarterback to another player is extremely difficult. The goal of the Steroscopic Targetting Project was to increase the accuracy of the QB and eventually complete more passes in a game setting. This project was implemented by installing an additional camera on the QB robot (bringing the total to 2 cameras). When the QB needed to throw a pass, both cameras would then track the receiving robot by searching for a color block in the field of view. The relative position from each camera is then used to determine the acual position of the receiving robot from the QB. This project is currently being expanded upon to allow for computer vision capabilities for other robots on the team

Malware Detection Project

Data Science

Malware is often transported onto personal devices through malicious links. The goal of the Malware Detection Project was to develop a set of classifiers capable of utilizing known information on given links to determine whether the link was malicious or benign. This project used the dataset found here to train and test the classifiers based on features such as URL length, IP Address of the host, geographic location of the host, length of the JavaScript code of the webpage, etc. The classifiers developed include multiple KNN Classifiers with varied K sizes, a Random Forrest Classifier, an ID3 Decision Tree, and a Convolutional Neural Network.
